# round_div

Calculate the division of a by b with rounding

## Arguments

- `a` - The dividend
- `b` - The divisor

## Returns

- `u128` - The result of the division with rounding (e.g., 5/3 = 2, 7/3 = 2, 8/3 = 3)

pub fn round_div(a: u128, b: u128) -> u128
